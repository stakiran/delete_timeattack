https://stakiran.github.io/delete_timeattack/

- <kbd>Delete</kbd>: Start game and delete `■`
- <kbd>Space</kbd>: Submit
- <kbd>Ctrl + X</kbd>,<kbd>Ctrl + V</kbd>: pasting `□` For miss-deletion
